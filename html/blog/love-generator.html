Finding 'LOVE' 💗

Searching for early strategy computer games, I came across David H. Ahl’s 1973 book: “BASIC Computer Games”.

In his book he compiles different early text-based computer games written in the BASIC programming language. This gave me the idea to re-implemented some programs using a modern programming language, in my case JavaScript.

The 'LOVE' program reproduces Robert Indiana's "Love" artwork on the console using a message of your choice. You can try it yourself using the p5.js canvas at the top of this page! The source code is available on github.
